User Manual

Using the Classifier

From the drop-down menu of classification types, select the type of classification you would like to use.

Drag and drop your files into the box provided, or click the box to browse your device for the files. Accepted file types are: .jpg, .jpeg, .png, .gif, .zip, .tar, .tar.gz. The box only accepts up to 10 files, but you can use one of the compressed file types to upload more than 10 images/files. If you upload a wrong file, simply click the "Remove file" link underneath to remove it.

Now, just click the “Upload and Classify!” button to get the result. To easily copy the output, simply click the “Copy results to clipboard” button.
