Galaxy Classification Tool

User manual

Getting In

To use the website as a regular user it is relatively simple, load up the webpage and enter the password that can be obtained through Kenji (the owner) either in person or by the email address provided (note that this password is subject to change as the owner sees fit).

Using Classifier

Once you have entered the password you are onto the main webpage. From here there is a drop-down menu of classification types, click here and select the type of classification you would like to use.

Drag and drop your images into the box provided or by clicking inside the box a menu pops up allowing you to select the files you wish to upload (within the limit), if you upload the wrong image then beneath the image when its uploaded there is a remove button to get rid of the image. You can only upload images that are in the formats listed and can only upload compressed files of the formats listed.

Now you just click the “Classify!” button to get the result. If multiple images are uploaded then there will be multiple results provided, if there are more than 10 then a text file can be downloaded where each line contains the results of the respective galaxy. Press the “Copy results to clipboard” button to save the output to your clipboard.
